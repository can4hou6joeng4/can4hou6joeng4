# Contributing

This repository powers the GitHub profile README for `can4hou6joeng4`.

Small fixes are welcome, especially typo fixes, broken-link reports, project metadata updates, and suggestions that make the profile clearer for open-source visitors.

## Guidelines

- Keep the README focused on current public work, not private or sensitive information.
- Prefer concise project descriptions that explain user value before implementation detail.
- Verify links before opening a pull request.
- Keep badge and image usage restrained so the profile remains readable on mobile.

## Pull Requests

For a pull request, include:

- What changed.
- Why the change improves the public profile.
- Any links or screenshots needed to review the change.

## Commit Messages

Use concise semantic commit messages in this format:

```text
type: 中文描述
```

For this profile repository, common types are `docs`, `style`, and `chore`.
