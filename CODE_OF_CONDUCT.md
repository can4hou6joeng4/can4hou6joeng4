# Code of Conduct

This profile repository follows a simple standard: be respectful, constructive, and specific.

## Expected Behavior

- Use welcoming and professional language.
- Focus feedback on the content, links, or project presentation.
- Assume good intent while still being precise about problems.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Publishing private information without consent.
- Spam, phishing links, or unrelated self-promotion.

## Enforcement

Maintainers may close, delete, or report comments and pull requests that violate this standard.
